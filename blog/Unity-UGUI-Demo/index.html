准备工作

  • 创建一些重要的文件夹

    Resources文件夹用于动态加载

    StreamingAssets文件夹用于存放Json数据文件

    ArtRes文件夹存放艺术资源

  • 导入Json相关脚本

    导入Json脚本后会自动创建Scripts文件夹

  • 将UI资源导入ArtRes文件夹

  • 创建Canvas画布对象

    1. 将场景名改为LoginScene用于登入场景
    2. 随便创建一个UI对象，然后将其删除，只保留它的父对象Canvas
    3. 如果有3D模型或粒子特效要显示在前面，要将Canvas组件中的Render Mode 改为 Camera模式
      1. 然后在场景当中 右键创建一个摄像机，然后将其Culling Mask参数只渲染UI层。再将其Clear Flags参数只渲染Depth only当前层级。最后移除该摄像机上的Audio Listener组件(该组件只能有一个)
      2. 选中主摄像机，让其Culling Mask参数不渲染UI层。再将其Clear Flags参数设置为Solid Color,再将背景色设置为纯黑色
      3. 将UI摄像机关联到摄像机模式上，摄像机渲染模式相关内容即可完成
    4. 拼面板之前要设置Canvas Scaler分辨率自适应相关内容
      1. 将UI Scale Mode 设置为Scale With Screen(缩放模式)
      2. 打开UI资源，打开大图素材，即可看到素材的分辨率
      3. 将Refernce Resolution(参考分辨率)设置为该素材分辨率
      4. 若是横屏游戏，Match要拖动到1。竖屏游戏，Match为0
    5. Canvas准备完毕，之后的面板在Canvas上拼凑即可

注意：以下是知识点补充

Canvas

Canvas是画布，它用于渲染自己的所有UI子对象，也就是说我们一般会在画布上进行拼凑面板，一般一个场景只需要一个画布即可

Render Mode

Canvas有三种渲染模式，常见的Overlay(覆盖模式),特点是将UI始终显示在前面。

还有Camera(摄像机模式),特点是将3D模式或粒子特效显示在UI前面

Canvas Scaler

CanvasScaler是 画布缩放控制器，它用于分辨率自适应组件

若想要分辨率自适应，就要选择缩放模式，画布随屏幕尺寸进行缩放大小

面板基类

用面板基类来描述子类面板的一些共同特点

  • 显示面板，即动态创建
  • 隐藏面板，即动态删除
  • 显示时淡入，隐藏时淡出

注意：知识点补充

通过为UI对象添加Canvas Group组件

其Alpha参数可以实现淡入淡出效果，Interactable参数可以实现UI对象是否禁用

思路：当面板之间切换时，更改Alpha的值

让Alpha从0~1，可以实现淡入显示面板的功能

让Alpha从1~0，可以实现淡出隐藏面板的功能

UI管理器

因为我们的UI面板都是动态的创建和动态删除的，也就是说面板一开始并没在场景上，并不能通过自己来控制显隐

所以我们为其单独的制作一个UI管理器，将其制作成一个单例模式，用于整体的管理 显示、隐藏、获取面板等逻辑
